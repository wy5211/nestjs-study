## 功能概览

- [x] 数据库：支持 [TypeORM](https://www.npmjs.com/package/typeorm) 与 [Mongoose](https://www.npmjs.com/package/mongoose)
- [x] 数据播种（Seeding）
- [x] 配置中心（[@nestjs/config](https://www.npmjs.com/package/@nestjs/config)）
- [x] 邮件发送（[nodemailer](https://www.npmjs.com/package/nodemailer)）
- [x] 邮箱注册与登录
- [x] 第三方登录（Apple、Facebook、Google）
- [x] 管理员与普通用户角色
- [x] 国际化/多语言（I18N）（[nestjs-i18n](https://www.npmjs.com/package/nestjs-i18n)）
- [x] 文件上传（本地与 Amazon S3）
- [x] Swagger 文档
- [x] E2E 与单元测试支持
- [x] Docker 支持
- [x] CI（Github Actions）
